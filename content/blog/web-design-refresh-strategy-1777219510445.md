# 🎯 Web制作会社のサイト刷新
## 市場調査による現状分析

> **旧サイトの3大課題**
> - 情報過多による認知負荷
> - モバイル最適化の不足
> - CTA（Call-to-Action）までの距離が遠い

---

## 📊 業界No.1の成功パターン

### Stripe（決済系UI/UX）から学ぶ
```
✅ ホワイトスペースの積極活用
✅ 1スクロール = 1メッセージの原則
✅ 数字・統計で信頼を可視化
```

### Figma（デザインツール）から学ぶ
```
✅ 段階的な情報公開（スクロール誘引）
✅ インタラクティブなホバー効果
✅ ユーザーの感情遷移を設計
```

## 🎨 刷新デザイン戦略

### 1️⃣ ファーストビュー（信頼構築フェーズ）

**背景**: グラデーション（白→グレー#F3F4F6）
**主視覚**: 大型タイポグラフィ + ミニマル幾何学図形

```markdown
### 「デジタル資産を、
### 信頼で変える」

📈 **実績シグナル帯**
- 👥 500+ 企業のパートナー
- ⚡ 平均成約率 340% UP
- 🏆 国際受賞歴 15年
```

**心理効果**: 数字の具体性は信頼度を即座に向上させます

---

### 2️⃣ CASE実装セクション（欲求喚起フェーズ）

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━
📸 ビジュアルケース
( 業界別・成果別にフィルタリング可能 )
━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔹 金融 x UI刷新
   → 申込数 +240% / コンバージョン率 +18%
   
🔹 SaaS x 体験設計
   → 継続率 89% / NPS +45
   
🔹 EC x パフォーマンス最適化
   → LCP 1.2秒達成 / 売上 +65%
```

**デザイン**: カード型3列レイアウト（モバイル時1列）
**インタラクション**: ホバー時に影と背景色が段階的に変化

---

### 3️⃣ プロセス透明化セクション（論理的納得フェーズ）

```
┌─ STEP 1: ヒアリング & 分析
│  └─ 市場調査・ユーザーリサーチ・競合分析
│
├─ STEP 2: 戦略策定
│  └─ ペルソナ設計・カスタマージャーニー・KPI定義
│
├─ STEP 3: デザイン & 開発
│  └─ プロトタイプ・ユーザーテスト・反復改善
│
└─ STEP 4: 検証 & 最適化
   └─ A/Bテスト・データ分析・継続支援
```

**デザイン**: タイムライン型の左右交互配置
**配色**: 各ステップにサテライト色（グラデーション）を付与

## 🎯 CTA戦略（最終行動誘引）

### プライマリCTA
```html
┏━━━━━━━━━━━━━━━━━━━━━━┓
┃ 📞 無料相談予約
┃ 限定30分 / 提案資料付き
┃ → 日程確認へ
┗━━━━━━━━━━━━━━━━━━━━━━┛
```
**色**: Emerald-600 (#059669)
**効果**: 余白と対比で視線が自動的に集中
**テキスト**: 「限定」「無料」が心理的ハードルを低下

### セカンダリCTA
```markdown
📧 メール相談（即時対応）
📄 資料DL（5分で概要把握）
```
**色**: グレー背景 + 深紺テキスト
**配置**: ホバー時に背景色がグラデーション化

## 🎓 採用理論の根拠

### 認知心理学
- **認知負荷理論**: 情報は3層に分割（ファースト→ケース→プロセス）
- **系列位置効果**: 最初と最後に重要情報を配置

### 行動経済学
- **社会的証明**: クライアント実績で信頼醸成
- **スカルシティ原理**: 「限定」で行動を誘発

### UXデザイン
- **ジェスタルト原則**: グループ化と整列で意味の認識
- **フィッツの法則**: CTAボタン配置で接近性を最大化
