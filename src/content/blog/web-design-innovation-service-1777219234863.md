## 🎯 刷新の3本柱

### 1️⃣ **配色戦略：信頼×プレミアム感**

```
プライマリ：Slate Navy (#0F172A)
セカンダリ：Amber Gold (#FBBF24)
アクセント：Cyan (#06B6D4)
ニュートラル：Modern Gray (#E2E8F0)
```

**心理的根拠**：
- 💎 ネイビーは金融・テック業界の信頼の色
- ✨ ゴールドは高級感と成功を象徴
- 🔵 シアンはイノベーションとアクティビティ

## 🎬 アニメーション戦略

✅ **採用すべき**：
- スクロールトリガー: 要素がビューポートに入ったら段階的フェードイン
- 視差スクロール: Hero背景とテキストのスピード差
- ホバーエフェクト: ケーススタディカード → 枠線拡大 + 背景暗化
- カウントアップアニメーション: 実績数字を0から目標値へ

❌ **回避すべき**：
- 自動再生動画（ユーザーストレス）
- 多数の点滅・フラッシュ（集中力低下）
- 速すぎるアニメーション（酔い感）
