## 🎯 採用する4つのデザイン要素

### ✨ 01. カラーシステム

> **主調色：深紺（#0F172A）** - 信頼感と高級感を同時実現

```
背景：#0F172A（ほぼ黒、でも冷たくない）
アクセント：#D4AF37（メタリックゴールド）→ボタン・リンク・アイコン
サポート：#F5F5F5（ホワイト98%）→テキスト・カード
ハイライト：#00D4FF（サイバーブルー）→CTA・アニメーション
```

---

### 🔤 02. タイポグラフィ戦略

**ヘッドライン：Sohne Regular 700-900**
- 幾何学的で現代的
- 大文字・サンセリフで権威感

**本文：Inter Regular 400-500**
- 画面での可読性最高クラス
- 数字の視認性（金融機関推奨フォント）

**見出し：Poppins SemiBold 600**
- ウォームで親しみやすい
- モバイルでも崩れない

## ⚡ 技術スタック（パフォーマンス重視）

```markdown
### 💻 フロントエンド
- **フレームワーク**：Next.js 14（App Router）
- **スタイリング**：Tailwind CSS 3.4
- **アニメーション**：Framer Motion 10 + GSAP
- **画像最適化**：next/image + WebP自動生成
- **動画**：VP9コーデック + 複数ビットレート

### 🔌 バックエンド
- **CMS**：Contentful（ヘッドレス）
- **ホスティング**：Vercel（エッジキャッシュ）
- **DB**：PostgreSQL（分析用）
- **監視**：Sentry + LogRocket

### 📈 SEO・マーケティング
- **メタ**：Next.js generateMetadata（動的生成）
- **構造化データ**：Schema.org（Organization/LocalBusiness）
- **計測**：GA4 + Hotjar（セッション録画）
- **コンバージョン**：Segment（CDP）で全接触点追跡
```
