# Web制作会社の革新的サイト刷新戦略

---

## 🎯 業界課題の解き方

> **従来のWeb制作サイトの問題点**
> - デザインで魅せているが、実績が不明確
> - 問い合わせまでの導線が長すぎる
> - モバイル対応が甘い
> - 企業文化が見えない

---

## 🚀 4つの刷新ポイント

### 1️⃣ ビジュアルアイデンティティの確立

```
配色戦略:
- Primary: #1F2937（ディープグレイ） → 信頼性
- Secondary: #7C3AED（パープル）→ 創造性
- Accent: #14B8A6（テール） → 行動喚起
- Background: #FAFAF9（ウォームホワイト）
```

**フォント選定**
- 見出し: `font-family: 'Hubot Sans', sans-serif` ← 技術企業の標準
- 本文: `font-family: 'Inter', system-ui` ← 国際的で読みやすい
- 数字: `font-variant-numeric: tabular-nums` ← データの信頼性演出

---

### 2️⃣ セクション構成の最適化

| セクション | 目的 | 配置 | 高さ |
|-----------|------|------|------|
| **ヒーロー** | 3秒で概念理解 | 最上部 | 60vh |
| **実績数値** | 信頼醸成 | スクロール25% | 30vh |
| **課題→解決策** | 共感と説得 | スクロール45% | 80vh |
| **クライアント事例** | 具体性 | スクロール70% | 90vh |
| **チーム紹介** | 人間味 | スクロール85% | 50vh |
| **CTA** | コンバージョン | フッター | 25vh |

---

### 3️⃣ インタラクション設計

```markdown
✨ **スクロールトリガーアニメーション**
- 数字のカウントアップ（0 → 最終値を3秒で）
- 実績カードの段階的フェードイン
- 背景の微妙なパララックス（-5%）

🎬 **ホバーエフェクト**
- ボタン: `transform: scale(1.02) + box-shadow深化`
- リンク: `text-decoration-line: underline + color shift`
- カード: `translate(0, -4px) + shadow elevation`

⚡ **ページロード最適化**
- First Contentful Paint: < 1.2s
- Largest Contentful Paint: < 2.5s
- Cumulative Layout Shift: < 0.1
```

---

### 4️⃣ モバイル優先の導線設計

```
┌─────────────────────────────┐
│  ヒーロー + CTA（目立つ）    │ ← 1タップ目
├─────────────────────────────┤
│  数字で信頼性の可視化        │
├─────────────────────────────┤
│  「こんな課題ありませんか」  │ ← 2タップ目（共感）
├─────────────────────────────┤
│  実装例3-4件のみ（厳選）     │
├─────────────────────────────┤
│  「まずは無料相談」ボタン   │ ← 3タップ目（CV）
└─────────────────────────────┘
```

## ✅ 実装のポイント

<mark>**絶対に守るルール**</mark>

1. **過度なアニメーション排除** → 読み込み速度の敵
2. **フォントファイルの最小化** → woff2形式のみ使用
3. **色数の制限** → 5色以内で統一感を保つ
4. **余白を味方に** → 詰め込み感は信頼低下に直結
5. **A/Bテスト前提** → CTA色・配置は仮説検証で決定
