## 🎯 3つのコアデザイン戦略

### 1️⃣ **ダークモード×プレミアムカラーパレット**

```
背景色: #0A0E27（深紺黒）
アクセント: #00D9FF（サイバーシアン）
セカンダリ: #FF006E（ホットピンク）
テキスト: #F5F5F5（オフホワイト）
```

**理由**
- Apple、Figma、Stripe等のプレミアムSaaS企業が採用
- 高い視認性 + 目の疲労低減
- 最新鋭・テック企業というイメージを自動付与

---

### 2️⃣ **フォント戦略：モダン二層構成**

| 用途 | フォント | 理由 |
|------|---------|------|
| **見出し** | `Sohne` / `Montserrat Bold` | Figma, Notion, Substack採用。洗練かつ力強い |
| **本文** | `Inter` / `-apple-system` | Google推奨。可読性最高峰 |
| **ロゴ** | `Space Mono Bold` | モノスペース系で技術感を演出 |

---

### 3️⃣ **セクション構成（黄金順序）**

```
┌─────────────────────────────┐
│ 1. ヒーロー動画セクション    │ ← フルスクリーン没入
├─────────────────────────────┤
│ 2. 3つの実績数字（大きく）   │ ← 信頼の可視化
├─────────────────────────────┤
│ 3. 厳選ケーススタディ 4件     │ ← 最高傑作のみ
├─────────────────────────────┤
│ 4. 認定資格・受賞ロゴ群      │ ← 権威性確保
├─────────────────────────────┤
│ 5. チーム紹介（顔写真重視）  │ ← 人間性・信頼感
├─────────────────────────────┤
│ 6. 料金表・パッケージ         │ ← 透明性
├─────────────────────────────┤
│ 7. CTA群（3段階最適化）      │ ← コンバージョン
└─────────────────────────────┘
```

## 🎬 動画背景戦略

> 📹 **推奨仕様**
> - 形式：MP4（H.265）/ WebM
> - 長さ：8〜12秒ループ
> - ファイルサイズ：3MB以下
> - コンテンツ：コード実装 or デザイン作業風景 or 完成品の変形アニメーション
> - オーバーレイ：#0A0E27半透過 60%（テキスト可読性保証）

## 🚀 競合比較優位性

| 要素 | 他社 | 当サイト |
|------|------|--------|
| **デザイントレンド** | グラスモーフィズム（既に古い） | ニューモーフィズム + ダークモード |
| **ポートフォリオ配置** | 下の方 | 最上部（認識段階で影響） |
| **アニメーション** | 静止 or 過度 | 戦略的マイクロイン |
| **モバイル対応** | 簡素 | フル最適化 + タッチインタラクション |
| **メタデータ** | 不十分 | OGP完全実装 + Schema.org構造化 |
