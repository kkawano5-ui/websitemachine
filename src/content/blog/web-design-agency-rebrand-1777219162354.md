## 🎯 推奨デザインシステム

### 配色パレット

```
🔵 プライマリ: #0F172A（深紺）
⚫ セカンダリ: #1F2937（炭素グレー）
✨ アクセント: #3B82F6（エレクトリックブルー）
⚪ ニュートラル: #F8FAFC（ホワイト）
```

> **理由**: 深紺は「信頼・プロフェッショナル」の心理学的効果。エレクトリックブルーはFigma/Dribbble等の新興企業が採用し、「イノベーション」と同義化している。

## 🔤 タイポグラフィ戦略

### 推奨フォント

**見出し系**: 
- `font-family: "Inter", "Helvetica Neue", sans-serif;`
- `font-weight: 700;`
- `letter-spacing: -0.02em;`

**本文系**:
- `font-family: "Inter", "Segoe UI", sans-serif;`
- `font-size: 16px;`
- `line-height: 1.6;`

> 💡 **Interの採用理由**: Apple/Google/Meta/GitHub全て採用済み。キリンビールのブランドリニューアルでも選定。国際感と最先端感を同時実現。

## ⚡ UX動線の最適化

### 推奨ユーザージャーニー

```
① Hero で視線を掴む ← 派手なアニメーション
   ↓（スクロール継続率: 85%目標）
② Value で「できる会社」と認識 ← データポイント
   ↓（信頼度 UP）
③ Case Study で具体例を確認 ← 『あ、この業界も実績ある』
   ↓（購買意欲 100%に）
④ Contact フォーム入力 ← 最小項目（企業名/メール/電話）
   ↓
⑤ Thank You ページ で次の行動指示 ← 営業へのハンドオフ
```

### クリック補助線

- **パンくずリスト**: 不要（シングルページスクロール推奨）
- **ナビゲーション**: スティッキーヘッダー（背景: 透過 → スクロール後に 90% 透過度）
- **外部リンク**: すべて `target="_blank"` + icon（↗︎）

## 🎬 アニメーション・インタラクション仕様

### スクロール駆動アニメーション

```css
/* Framer Motion / AOS.js 推奨 */
.fade-up {
  animation: fadeInUp 0.8s ease-out;
}

@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(40px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

/* ホバーエフェクト */
.cta-button:hover {
  box-shadow: 0 20px 60px rgba(59, 130, 246, 0.4);
  transform: translateY(-2px);
  transition: all 0.3s cubic-bezier(0.34, 1.56, 0.64, 1);
}
```

/* デフォルト: dark mode */
